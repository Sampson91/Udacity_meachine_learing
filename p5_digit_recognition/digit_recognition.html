问题 3¶

_你最终的模型架构是什么样的？（什么类型的模型，层数，大小, 如何连接等） 回答：

该模型采用keras搭建的10层卷积升神经网络，第一层为input输入层对（28,140,1）的图像进行输入，（2,4,6）层为（32,64,128）个核的尺寸为3×3的卷积层，(3,5,7）为2×2的池化层，第八层为展开层，把数据转换为一维，第九层为dropout层。第十层为5个结果的输出层。

其中网络中各个层的技术实现方式为：
    1.在网络中卷积层采用3×3像素的尺寸的窗口在图像中进行滑动对每个3×3的窗口根据学习的权重进行加权求和，最后返还一个经过卷积的图片;获得的图片通过relu函数f(x)=max(0,x)来将所有负值像素替换为0。
    2.由于卷积层获取的图片尺寸没有发生变化，而为了减少模型的复杂度，通过压缩图片来实现即通过神经网络中下采样来稀疏参数，该模型采用最大值池化Max_pool方法通过2×2的像素中选择最大的值作为下一层传输。。
    3.Dropout层是指在模型训练时随机概率让网络某些隐含层节点的权重不工作，不工作的那些节点可以暂时认为不是网络结构的一部分，但是它的权重得保留下来（只是暂时不更新而已），因为下次样本输入时它可能又得工作了。
    4.对构建好的模型通过梯度下降的方法学习卷积核的权重，对于梯度下降中各个权重参数有些维度变化快，有些维度变化慢；有些维度是负的斜坡，有些维度是正的斜坡采用相同的Learning Rate是不科学的，比如有的参数可能已经到了仅需要微调的阶段，但又有些参数由于对应样本少等原因，还需要较大幅度的调动。理想的方案是根据参数每个维度的变化率，采用对应的Learning Rate。Adadelta优化方法为每个参数计算自适应的Learning Rate，给不经常更新的参数以较大的Learning Rate， 给经常更新的参数以较小的Learning Rate并且Adadelta与其它优化算法相比解决了1）学习率是单调递减的，训练后期学习率非常小。2）其需要手工设置一个全局的初始学习率。3）更新W时，左右两边的单位不同问题。
    5.Softmax回归模型，该模型是logistic回归模型在多分类问题上的推广，在多分类问题中，类标签y可以取两个以上的值。 Softmax回归模型对于诸如MNIST手写数字分类等问题是很有用的，该问题的目的是辨识10个不同的单个数字对于给定的测试输入 x，我们想用假设函数针对每一个类别j估算出概率值 p(y=j | x)。也就是说，我们想估计 x 的每一种分类结果出现的概率。因此，我们的假设函数将要输出一个k 维的向量（向量元素的和为1）来表示这k个估计的概率值，也就是softmax把一个k维的real value向量（a1,a2,a3,a4….）映射成一个（b1,b2,b3,b4….）其中bi是一个0-1的常数，然后可以根据bi的大小来进行多分类的任务，如取权重最大的一维。

练习：计算你的模型准确率¶

我们刚才得到了模型每个数字的准确率，现在让我们来计算整体准确率，按照完全预测正确数字序列的标准来计算。

比如 1,2,3,10,10 预测成了 1,2,10,10,10 算错，而不是算对了80%。

def evaluate(model,X_test, y_test):
    # TODO: 按照错一个就算错的规则计算准确率
    y_test_predict = model.predict(X_test)
    y_test_predict = np.array(y_test_predict)
    y_test_predict = np.argmax(y_test_predict, axis = 2)
    y_test = np.argmax(y_test,axis = 2)
    y_test_predict = y_test_predict.T
    y_test = y_test.T
    number = 0
    for i in range(len(y_test_predict)):
        if (y_test_predict[i] == y_test[i]).all():
            number += 1
    accuracy = float(number)/len(X_test)
    return accuracy
